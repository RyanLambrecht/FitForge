FitForge

FitForge is a web application that allows users to track their workouts and progress. Users can create an account, log in, search for and create workouts. Workouts can be saved and viewed later. Users can also view their workout history and see their progress over time. You are also able to use fitForge to track you Food intake and search for nutrition information on food items.

Built using Django
